# C.5.2.3 bldg:GroundSurface

## C.5.2.3 bldg:GroundSurface

bldg:GroundSurfaceは、底面を記述するために使用する地物型である。LOD2で底面を記述する場合に、
その高さは、 図C-25に示す二つの方法のいずれかにより決定する。建築物に地階が存在し、その最も低い高さが得られる場合にはその高さを使用する。高さの情報が得られない場合には、bldg:GroundSurfaceの水平位置と地形モデルとの交線上で最も低い高さを取得し、この値を使用する。
