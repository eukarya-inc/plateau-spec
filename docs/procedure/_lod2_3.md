# E.2.3.3 LOD2

## E.2.3.3 LOD2

交通（鉄道）モデル（LOD2）では、鉄道の形状を面として表現する。このとき、tran:Railwayは、横断構成要素であるtran:TrafficAreaに分解される。tran:Railwayの空間属性は、これを構成するtran:TrafficAreaの空間属性の集まりとなる。

**表 E-7 — E-5: LOD2 tran:Railwayの空間属性一致条件**

| ID | `/req/rwy/5` |
| --- | --- |
| 主題 | 妥当な交通（鉄道）データセットの要件 |
| 分類 | 要件分類E-1: 妥当な交通（鉄道）オブジェクト |
| 条文 | LOD2におけるtran:Railwayの空間属性は、これを構成するtran:TrafficAreaの空間属性の集まりと一致しなければならない。 |

交通（鉄道）オブジェクトは、交通（鉄道）モデル（LOD2）の定義に従ったものでなければならない。

**表 E-8 — E-6: 鉄道のLOD2形状定義**

| --- | --- |

| 分類 | 要件分類E-1: 妥当な交通（鉄道）オブジェクト |
| 条文 | 鉄道のLOD2の形状は、交通（鉄道）モデル（LOD2）の定義に従う。 |
