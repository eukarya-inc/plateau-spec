# E.3.1.3.3 作業手順（TrafficArea）

## E.3.1.3.3 作業手順（TrafficArea）

- 交通（鉄道）モデル（LOD1）に加えて、航空写真又は平面図を参考に道床をポリゴンで取得する。このとき、道床とレールに囲まれた範囲が重複する範囲はレールに囲まれた範囲を優先し、道床のポリゴンは取得しない。道床はレール又はまくらぎを支持し、荷重を路盤に分布する軌道の部分のことを指す。図E-6 — 線路の構造
- 道床、レールに囲まれた範囲及び軌道中心線を、それぞれTrafficAreaに区分する。
