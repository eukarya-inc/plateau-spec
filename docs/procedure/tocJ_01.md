# J.1 概要

## J.1 概要

本付属書では、「3D都市モデル標準製品仕様書」（以下、「標準製品仕様書」という）に適合したデータ製品について、「災害リスクモデル」に従って作成されるべき都市オブジェクト（以下、「災害リスクオブジェクト」という）が満たすべき要件と、妥当な災害リスクオブジェクトを作成するための標準的な作業手順を示す。
