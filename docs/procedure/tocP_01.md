# P.1 概要

## P.1 概要

本付属書では、「3D都市モデル標準製品仕様書」（以下、「標準製品仕様書」という）に適合したデータ製品について、「地下埋設物モデル」に従って作成されるべき都市オブジェクト（以下、「地下埋設物オブジェクト」という）が満たすべき要件と、妥当な地下埋設物オブジェクトを作成するための標準的な作業手順を示す。

**表 P-1 — P-1: 妥当な地下埋設物オブジェクト**

| ID | `/req/unf` |
| --- | --- |
| 対象の種類 | 妥当な地下埋設物オブジェクトの要件 |
