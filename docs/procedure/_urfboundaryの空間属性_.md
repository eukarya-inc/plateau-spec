# K.3.1 urf:Boundaryの空間属性

## K.3.1 urf:Boundaryの空間属性

urf:Boundaryは、urf:Zoneの境界となる。そのため、urf:Boundaryの空間属性（gml:MultiCurve）は、urf:Zoneの空間属性（gml:MultiSurface）を構成する曲線と一致しなければならない。
