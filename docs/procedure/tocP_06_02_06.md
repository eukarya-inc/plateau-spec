# P.6.2.6 管路及び函渠の外径の推定について

## P.6.2.6 管路及び函渠の外径の推定について

管路の外径が不明な場合は、管路の管種及び内径から外径を推定してよい。下水道管の外径の推定には日本下水道協会規格（JSWAS）を用いる。
