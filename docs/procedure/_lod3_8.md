# L.2.3.4 LOD3

## L.2.3.4 LOD3

橋梁モデル（LOD3）では、橋梁の形状を、主要な部分の外形を構成する特徴点から構成する面の集まり又はこれらの面を境界面とする立体として表現する。

**表 L-10 — L-6: 橋梁のLOD3形状定義**

| ID | `/req/brid/6` |
| --- | --- |
| 主題 | 妥当な橋梁データセットの要件 |
| 分類 | 要件分類L-1: 妥当な橋梁オブジェクト |
| 条文 | 橋梁のLOD3の形状は、橋梁モデル（LOD3）の定義に従う。 |
