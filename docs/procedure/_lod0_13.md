# M.2.3.1 LOD0

## M.2.3.1 LOD0

トンネルモデル(LOD0)では、トンネルの形状を点、線又は面として記述する。このとき、トンネルオブジェクトは、トンネルモデル（LOD0）の定義に従ったものでなければならない。

**表 M-4 — M-2: トンネルのLOD0形状定義**

| ID | `/req/tun/2` |
| --- | --- |
| 主題 | 妥当なトンネルデータセットの要件 |
| 分類 | 要件分類M-1: 妥当なトンネルオブジェクト |
| 条文 | トンネルのLOD0の形状は、トンネルモデル（LOD0）の定義に従う。 |

トンネルモデル（LOD0）の取得方法は、数値地形図に従う。
