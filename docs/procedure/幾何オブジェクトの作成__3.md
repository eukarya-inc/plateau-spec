- 交通（広場）モデル（LOD3.1）は、交通（道路）モデル（LOD3.1）を参照して使用することを基本とする。交通（道路）モデル（LOD3.1）を参照して使用する場合は、改めて車線の区分は不要である。交通（道路）モデル（LOD3.1）が存在しない場合は、MMS点群等から道路部分を三次元図化により取得し、その後に車線の区分を行う。図G-10 — 車線の区分のイメージ
- 広場の範囲のうち、a）で作成した道路部分以外をMMS点群等から三次元図化により作成する。このとき、a）で作成した道路部分と接合が取れるように調整する。
