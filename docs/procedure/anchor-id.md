## 1.2 引用規格等

本書が引用する規格・仕様・マニュアルを以下に示す。3D都市モデルの整備に際しては、これらの文献を理解することが前提となる。
引用文献のうちで、版の記載があるものは、その版を適用し、その後の改正版（追補を含む）は適用しない。版の記載がないものは、その最新版（追補を含む）を適用する。 
また本書において、「i-UR」は、特段の指定がない限り、 [i-UR 3.2](https://www.chisou.go.jp/tiiki/toshisaisei/itoshisaisei/iur/index.html) を指す。また、「CityGML」についても同様に [OGC 12-019](https://www.ogc.org/standards/citygml) を指す。
