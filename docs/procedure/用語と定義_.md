# 1.4.1 用語と定義

## 1.4.1 用語と定義

### 1.4.1.1 地物

実世界の現象の抽象概念。
なお地物は、型又はインスタンスとして現れる。地物型又は地物インスタンスという用語は、いずれか一方だけを意味するときに使うことが望ましい。

［出典： ISO 19101]

### 1.4.1.2 地物型

地物の論理的な構造の定義。

### 1.4.1.3 都市オブジェクト

地物型に基づいて作成されたデータ。

### 1.4.1.4 幾何型

幾何形状の論理的な構造の定義。

### 1.4.1.5 幾何オブジェクト

幾何型に基づいて作成されたデータ。

### 1.4.1.6 空間属性

地物の空間的な特性。空間属性は、地物型と幾何型又は都市オブジェクトと幾何オブジェクトを、紐づける。
