# P.5.1 繰り返しオブジェクトの座標の単位

## P.5.1 繰り返しオブジェクトの座標の単位

地下埋設物モデルが採用する平面直角座標系の軸の単位はメートル（m）であることから、プロトタイプモデルの座標及び変換行列の単位もメートル（m）でなければならない。

**表 P-10 — P-8: プロトタイプモデルの単位条件**

| ID | `/req/unf/8` |
| --- | --- |
| 主題 | 妥当な地下埋設物データセットの要件 |
| 分類 | 要件分類P-1: 妥当な地下埋設物オブジェクト |
| 条文 | プロトタイプモデルの座標及び変換行列の単位はメートル（m）とする。 |
