# 4.13.1.4.1 その他の構造物モデル（LOD3）の概要

## 4.13.1.4.1 その他の構造物モデル（LOD3）の概要

その他の構造物モデル（LOD3）では、その他の構造物の形状を、主要な部分の外形を構成する特徴点から構成する面を境界面とする立体として表現する。

LOD3は、構造上不可欠ではない付属物（手すり、柵、構造物と一体ではない階段）の表現有無によりLOD3.0及びLOD3.1に区分する。
