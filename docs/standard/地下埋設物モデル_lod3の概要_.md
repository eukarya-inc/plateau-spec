# 4.15.1.4.1 地下埋設物モデル（LOD3）の概要

## 4.15.1.4.1 地下埋設物モデル（LOD3）の概要

地下埋設物モデル（LOD3）では、地下埋設物の形状を、詳細な面の集まりとして表現する。

地下埋設物モデル（LOD3）の取得イメージを表4-691に示す。
