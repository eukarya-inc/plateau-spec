# 7.2.1.1 ファイル単位

## 7.2.1.1 ファイル単位

ファイル単位は、表7-2に示す標準製品仕様書に示す応用スキーマの単位、かつ、JIS X0410において
定められた地域メッシュの単位を基本とする。一つのファイルには、同一の空間参照系のオブジェクトのみを含む。

地域メッシュは、第2次地域区画（統合地域メッシュ、一辺の長さ約10km）又は基準地域メッシュ（第3次地域区画、一辺の長さ約1km）とする。

| 土地利用 | ↑ |
| 水部 | ↑ |
| 土砂災害警戒区域 | ↑ |

| その他の区域 | ↑ |
