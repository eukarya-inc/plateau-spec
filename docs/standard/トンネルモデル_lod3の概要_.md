# 4.12.1.4.1 トンネルモデル（LOD3）の概要

## 4.12.1.4.1 トンネルモデル（LOD3）の概要

トンネルモデル（LOD3）では、トンネルの形状を、立体により表現し、立体を構成する各境界面を地物として取得するとともに、トンネルの開口部やトンネルの外側に付いている付属物を地物として取得する。

トンネルモデル（LOD3）の取得イメージを表4-554に示す。
