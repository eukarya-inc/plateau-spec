# 4.3.1.2 交通（道路）モデル（LOD1）

## 4.3.1.2 交通（道路）モデル（LOD1）

## 4.3.1.2.1 交通（道路）モデル（LOD1）の概要

交通（道路）モデル（LOD1）では、道路の形状を面により表現する。

交通（道路）モデル（LOD1）の取得イメージを表4-136に示す。

**表 4-136 — 交通（道路）モデル（LOD1）の取得イメージ**

|  | LOD1 |
| --- | --- |
| 取得例 | ![Image](images/toc4_03_01_02_img001.png) |
| 説明 | 道路縁により囲まれた範囲を面として取得し、以下の場所で区切る。交差部（四差路、多差路及び三差路）道路構造（トンネル、橋梁）が変化する場所位置正確度や取得方法が変わる場所高さは0とする。 |
