# 4.26 施設管理の応用スキーマ

## 4.26 施設管理の応用スキーマ

施設を管理するために必要な情報を定義する。これらは、都市オブジェクトの属性となる。

## 4.26.1 施設管理属性のLOD

施設管理属性はそれ自身で空間属性をもたないため、LODは定義しない。
