# 4.2.4.2.15 IfcElementCompositionEnum

## 4.2.4.2.15 IfcElementCompositionEnum

**表 4-120**

| 列挙型 | IfcElementCompositionEnum |
| --- | --- |
| 値 | 説明 |
| COMPLEX | 通常の単一の建物であればELEMENTを設定 |
| ELEMENT | 複数の建物から構成される複合建物の場合はCOMPLEXを設定 |
| PARTIAL | 部分的な空間を表現している建物の場合はPARTIALを設定 |
