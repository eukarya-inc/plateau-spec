# 4.18.1.1.1 地形（LOD0）の概要

## 4.18.1.1.1 地形（LOD0）の概要

地形モデル（LOD0）では、地形を点又は線により表現する。

地形モデル（LOD0）の取得イメージを表4-767に示す。
