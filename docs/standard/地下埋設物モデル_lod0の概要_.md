# 4.15.1.1.1 地下埋設物モデル（LOD0）の概要

## 4.15.1.1.1 地下埋設物モデル（LOD0）の概要

地下埋設物モデル（LOD0）では、地下埋設物の形状を、点及び線により表現する。

地下埋設物モデル（LOD0）の取得イメージを表4-685に示す。

**表 4-685 — 地下埋設物モデル（LOD0）の取得イメージ**
