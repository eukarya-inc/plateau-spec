# 4.4.1.2 交通（鉄道）モデル（LOD1）

## 4.4.1.2 交通（鉄道）モデル（LOD1）

## 4.4.1.2.1 交通（鉄道）モデル（LOD1）の概要

交通（鉄道）モデル（LOD1）では、鉄道の形状を面により表現する。

交通（鉄道）モデル（LOD1）の取得イメージを表4-180に示す。
