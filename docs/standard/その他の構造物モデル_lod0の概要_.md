# 4.13.1.1.1 その他の構造物モデル（LOD0）の概要

## 4.13.1.1.1 その他の構造物モデル（LOD0）の概要

その他の構造物モデル（LOD0）では、構造物の形状を、点、線、又は面により表現する。

その他の構造物モデル（LOD0）の取得イメージを表4-606に示す。

**表 4-606 — その他の構造物モデル（LOD0）の取得イメージ**
