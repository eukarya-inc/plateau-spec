# 4.15.1.2.1 地下埋設物モデル（LOD1）の概要

## 4.15.1.2.1 地下埋設物モデル（LOD1）の概要

地下埋設物モデル（LOD1）では、地下埋設物の形状を、立体により表現する。

地下埋設物モデル（LOD1）の取得イメージを表4-687に示す。
