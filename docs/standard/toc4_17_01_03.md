# 4.17.1.3 植生モデル（LOD2）

## 4.17.1.3 植生モデル（LOD2）

## 4.17.1.3.1 植生モデル（LOD2）の概要

植生モデル（LOD2）では、植生の形状を、面の集まり又は立体により表現する。

植生モデル（LOD2）の取得イメージを表4-748に示す。

**表 4-748 — 植生モデル（LOD2）の取得イメージ**

|  | LOD2 | ← |
| --- | --- | --- |
| 取得例 | ![Image](images/toc4_17_01_03_img001.png) | ![Image](images/toc4_17_01_03_img002.png) |
| 説明 | 樹冠と樹幹をそれぞれ簡略化した立体 a) を組み合わせた立体として表現する。 | 植被の表層の高さを取得し、比高3m以上を再現した立体として表現する。モデルは特徴点を結ぶ三角網から構成される面を立体として表現する。 |
|  | ← | ← |

簡略化した立体とは、楕円体、球体、円錐、角錐、角柱、円柱などの単純な立体図形とする。
