# 4.5.1.1.1 交通（徒歩道）モデル（LOD0）の概要

## 4.5.1.1.1 交通（徒歩道）モデル（LOD0）の概要

交通（徒歩道）モデル（LOD0）では、徒歩道の形状を線により表現する。このとき、徒歩道の形状の線は、ネットワーク（中心線）又は徒歩道縁のいずれかとする。

標準製品仕様書は、原則としてネットワークを採用する。ただし、数値地形図との互換性を保つために、徒歩道縁を選択できる。徒歩道縁を採用する場合、公共測量法準図式に従う。
